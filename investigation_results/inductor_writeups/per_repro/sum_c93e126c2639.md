## Gap Diagnosis

The oracle differs from Inductor by specializing the complete width-512
attention-backward row into one Triton program that computes mask/dropout,
softmax-gradient, row reduction, and FMA together while directly writing the
returned full tensor. Inductor cannot extract a substantially better path today
because the compiled repro is already a single fused kernel over the same full
tensor scope; for this captured shape the remaining cost is dominated by loading
the logits, upstream gradient, dropout mask, broadcast attention mask, and
writing the output rather than by a missing reduction decomposition. The class
is `BANDWIDTH_BOUND`, and the lower historical compile timing means this
artifact should not be used as a floor.
